# Neurofive ML Track - Task 3

## Model Selection and Evaluation

This project is completed as part of **Task 3** of the Neurofive Solutions Machine Learning Internship.

### Objective
Build a Logistic Regression model to predict Titanic passenger survival.

### Approach
- Loaded and explored the Titanic dataset.
- Handled missing values.
- Performed Exploratory Data Analysis (EDA).
- Encoded categorical features.
- Split the dataset into training and testing sets.
- Trained a Logistic Regression model.
- Evaluated the model using Accuracy Score and Confusion Matrix.

### Final Accuracy
**81.01%**

- Python
- Pandas
- Seaborn
- Matplotlib
- Scikit-learn
